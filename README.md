# AI-Powered Customer Support Ticketing System

## 📌 Project Overview
This project is a Salesforce-based AI-powered customer support ticketing system designed to automate and streamline customer issue management.

## 🚀 Features
- Ticket creation and tracking
- Priority management
- AI-based ticket categorization
- Status updates and resolution tracking
- Salesforce Admin configuration

## 🛠️ Technologies Used
- Salesforce Admin
- AI Agentforce
- Lightning Components
